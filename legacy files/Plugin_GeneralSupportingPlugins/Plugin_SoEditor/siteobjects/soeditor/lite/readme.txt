INSTALLATION NOTES
=-=-=-=-=-=-=-=-=-=
ż Extract this package into the root of the website you wish to install soEditor
  Lite in.

ż Copy soeditor_lite.cfm into a custom tag directory. (e.g. c:\cfusion\customtags\)
 
ż Open IE and access the documentation at
  http://{www.yoursite.com}/sitobjects/soeditor/lite/docs/index.cfm

ż If you extract the package to an alternative directory, you must change 2
  ColdFusion variables in docs/index.cfm to point to the new location. in order
  for the documentation to work correctly.
